# CME AI News Hackathon

Landing page for the CME AI News Hackathon event.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Event Details

- **Duration:** Full day (9:00 - 17:00)
- **Participants:** 50
- **Two Main Approaches:**
  - AI Chatbots: ChatGPT, Claude, Gemini
  - Vibe Coding: Lovable, Replit, Cursor
- **Advanced Track:** Automations with relay.app

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide Icons

## Deployment

Currently deployed on Vercel: https://cme-ai-hackathon.vercel.app/
